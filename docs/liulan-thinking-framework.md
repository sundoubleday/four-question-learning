# 刘澜的思维框架（女娲造人术·五元组）

> 来源：《学会如何学习：120堂认知升级课》（书1，约13.3万字）与《反向学习：学习力的五项修炼》（书2，约14.8万字），均为刘澜著，是"学习力五项修炼"思想的两个部分。本文件提炼其**看学习与成长的独特镜片**，而非具体方法步骤。证据均出自两书原文或章节。

---

## 1. 心智模型（Mental Models）

按女娲三重验证（跨域复现？生成力？排他性？）筛选，以下 6 个通过。

**① 反向学习镜片：成年人的学习首先是"减法"**
- 描述：成人不是空杯，而是一张写满错字、旧字、过时字的"字纸"；学习第一件事是擦除，而非累加。
- 证据：书2 导论/第一章"反向学习指有意遗忘头脑中已有的知识……成年人的大脑就像是一张字纸，已经积累了大量错误的、过时的、低效的知识"；同一章又用芒格的话佐证——"哪年你没有破坏一个你最爱的观念，那么你这年就白过了"。组织层面亦复现："组织也需要反向学习"（书2 第一章第五点）。
- 应用：先列"学习上的坏习惯清单"，用新习惯替代旧习惯（"用加法做减法"）。
- 局限：对白纸般的小孩，反向学习"不那么重要"（书2 第一章），正向学习优先。

**② 参考答案镜片：世上没有标准答案，只有参考答案**
- 描述：把一切权威（书、老师、自己）的输出当作可修改的参考，而非必须照搬的标准。
- 证据：书2 前言把成人心智分三阶段——"标准答案思维方式→参考答案思维方式1→参考答案思维方式2（连自己的答案也当参考答案）"；书2 又举德鲁克曾被马斯洛指出"激励人无标准答案"，德鲁克"有风度地承认自己错了"。
- 应用：读任何书都自问"这是我的参考答案吗"，而非"这是真理吗"。
- 局限：他自承"标准参考答案"（如"看情况"）在绝大多数情境管用、接近标准但仍仅供"参考"（书2 第二章策略二），故不可走向彻底相对主义。

**③ 模式化镜片：磁铁吸铁屑，而非散沙堆散沙**
- 描述：专家握"模式"（共性/整体/本质），新手只有"碎片"（个性/表象）。模式知识像磁铁，碎片知识像铁屑。
- 证据：书2 第一章"专家掌握模式知识，新手只掌握碎片知识"；同一节"如果你的'我知道'是模式知识，它就能像一块磁铁一样，帮助你吸收'我不知道'的碎片知识。而如果你的'我知道'只是碎片知识，那么它即使看起来很多，也只是一堆铁屑"。书1 第2课用"小狗摇尾巴讨人喜欢"说明：马/人不能照搬狗的经验，须找背后模式。
- 应用：学任何内容都问"背后的模式是什么"（书1 第2课、四问学习法）。
- 局限：建立"模式树"本身有门槛，对新手是"慢思考"的硬仗。

**④ 已知-未知依存镜片：学习力第一公式**
- 描述：你学到什么，取决于你已知道什么。公式是"学习 = 我知道 + 我不知道"。
- 证据：书1 第1课"我和A同学的'我知道'不一样，'我不知道'也就不一样……没有放之四海而皆准的好书"；书2 导论引奥苏贝尔"影响学习最重要的因素就是学习者已经知道了什么"，并称此为"关于学习最重要的道理"。
- 应用：选书/听课先评估自己的"我知道"，判断能否产生"我不知道"的碰撞。
- 局限：公式本身只是"最一般意义上"的描述，不告诉你具体该补哪块"我知道"。

**⑤ 聚焦少而深镜片：二八法则对抗"贪多求浅"的系统默认**
- 描述：有少数知识比其他知识重要得多；发散式（补短板、全面发展）必然学得多而浅，须聚焦。
- 证据：书2 第三章"有少数事情比其他事情更重要……二八法则"；同一章称贪多求浅是"我们大脑系统的默认设置，也就是卡尼曼的'快思考''系统一'"，须用"口诀"唤醒"系统二"。
- 应用：用聚焦口诀"对我来说，有少数知识比其他知识更重要。我要聚焦的少数知识是什么？"（书2 第三章）。
- 局限：聚焦专题需长期投入（"用5年成小专家，10年成大专家"），与人的本能相悖，极难坚持。

**⑥ 动词化/用镜片：用不起来的学习一定是假学习**
- 描述：学须落在习（用）上；"用"既是试金石，也是脚手架（学以致用 + 用以致学）。
- 证据：书1 第8课"用不起来的学习，往往是假学习……甚至可以说得更绝对一些，用不起来的学习，一定是假学习"；书2 称"卓有成效的学习者过动词化的人生"。
- 应用：对任何知识立刻问"我可以怎么用"（非"应该"、非"能够"，书2 第五章）。
- 局限：纯"为知识而知识"者，他承认"也行"，只是水平难高（书1 第8课）。

## 2. 决策启发式（Decision Heuristics，5-10条）

1. **如果**有人问你任何管理/学习问题 → **则**先答"看情况"，并准备回答"怎么看情况"（用洞见分类）。案例：书2 导论"管理者最重要的能力是什么？你可以回答：看情况"；"刘澜管理问题第一法则：所有的管理问题都只有一个标准答案——看情况"。
2. **如果**读到/听到一个观点 → **则**同时三问："如果对，对在哪？如果错，错在哪？更好的观点是什么？"（守破离）。案例：书2 第二章"这组口诀"对应互动学习的守、破、离，须"同时使用"。
3. **如果**要开始学习/刚经历失败 → **则**先问"我要改变什么？"（反向学习口诀）。案例：书2 第一章把失败（错失晋升、搞砸项目）作为反向学习触发点。
4. **如果**学到任何知识 → **则**问"我可以怎么用？"而非"我应该/能够怎么用"。案例：书2 第五章辨析"可以"暗示"你用得上且无唯一标准答案"。
5. **如果**面对标准答案 → **则**把它降级为参考答案，且把自己的答案也当参考答案。案例：书2 前言三阶段模型。
6. **如果**学高手/成功者经验 → **则**警惕幸存者偏差 + 非线性，追问"背后的模式是什么？对我适用吗？"。案例：书1 第3课卡尼曼公式"成功=能力+运气，巨大成功=多一些能力+多很多运气"，故不主张学谷歌/首富。
7. **如果**看到碎片（个人经验/小事） → **则**停下来找模式，勿照搬。案例：书1 第2课"狗摇尾巴"、第4课从"迟到"看出"是否尊重他人"的模式。
8. **如果**做迁移 → **则**用四问学习法（听想变用），从模式层次迁移，而非表层生搬。案例：书2 第五章学"海底捞发眼镜布"须先找到"服务意识"模式再迁移，而非照抄动作。
9. **如果**有人声称唯一正确方法 → **则**不盲从，"你读了有收获的书就是好书"，无放之四海皆准之法。案例：书1 第1课。

## 3. 表达 DNA

- **句式**：大量反问制造参与感——"你会怎么回答这个问题？""你有没有想过，对所有这些问题，'看情况'可能是更好的回答呢？"；短句、口语化（书1 前言明言由"语音点评"转写，"保留了一些口语化的特点"）。
- **自创术语/高频词**：四问、五项修炼、看情况、口诀、模式、参考答案、看家模式、知识磁铁/饭碗/钥匙、体温、天文学家、农民、守破离、深层迁移、反向学习。
- **招牌比喻**：磁铁-铁屑（模式vs碎片）、禅宗指月（手指≠月亮，五项修炼"指向同一个月亮的手指"）、乔布斯同款衣服（节省认知资源）、小狗摇尾巴（经验不可照搬）、小鱼问"水是什么"（隐性课程如空气）、齐美尔农场主子孙（聚焦深耕）、橡皮擦擦字（反向学习之难）。
- **节奏**：先抛结论再铺垫——"我先教你一个万金油式的回答：看情况"；爱用"好消息/坏消息"框架（如反向学习章"三个好消息和一个坏消息"）；用"公式"与"口诀"封装原理（学习=我知道+我不知道；成功=能力+运气）。
- **确定性姿态**：混合态——对**原理**极其笃定（"关于学习，如果你只能记住一个道理，那你应该记住这一个""用不起来的学习，一定是假学习"），对**应用**极度谦逊（"看情况""仅供参考"）。这是一种"原理确信、情境谦卑"的独特腔调，区别于普通博主的"三步速成"式绝对化。

## 4. 反模式（Anti-patterns）

刘澜明确反对：
- **标准答案思维**：把书/老师/自己当唯一正确。
- **碎片化学习**：学到一堆互不相关的"散沙"。
- **发散式学习/补短板/全面发展**：导致"学得多而浅"。
- **学用脱节**：分"学而不用"（危害更重）与"学而乱用"（表层生搬，如海底捞店长照搬发眼镜布）。
- **附属坏习惯**：被动学习、单人学习、害怕出错、重结果轻过程（皆由标准答案思维衍生）。
- **"天文学家"式学习**：纸上谈兵，像谈火星一样谈知识，不联系自身（书1 第104课、书2 第一章）。
- **照搬个人经验**：只学"一个样本"，忽略模式（书1 第2、3课）。

## 5. 诚实边界（Honest Boundaries）

- **对象前提**：五项修炼针对"已被学校隐性课程中毒的成年人"；对小孩反向学习不重要（书2 第一章）。
- **方法本身是参考答案**：他反复声明两本书"都是我给出的参考答案"，读者"不能生搬硬套"，连"看情况"也须配"怎么看情况"才不沦为空话。
- **洞见分类门槛高**：他承认洞见分类"是主观的、理想的"，现实事物是混合体，归入不易（书2 第二章）。
- **反向学习极难**：旧心智模式"像是刻在木头里的字，很难擦掉"（书2 第一章引凯恩斯"难的是清除旧观念"），且"负隅顽抗"，是漫长修炼。
- **聚焦反本能**：贪多求浅是系统1默认设置，需外力（口诀）才勉强唤醒系统2，多数人难坚持5–10年。
- **来源偏向**：其案例多来自企业管理者/创业者，"学习力"被他定义为"元能力"普适于所有人，但对儿童教育、纯学术训练未必直接适用。
- **概念仍在生长**：他坦承"体温"概念在书1写作中从"两种→三种→困惑的体温"不断变化，"这时，体温的概念似乎又跟前面不同"——部分框架尚未定型，不宜当封闭体系供奉。
