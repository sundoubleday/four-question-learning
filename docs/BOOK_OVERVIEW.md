## 一句话理解

刘澜这两本书不是教"怎么记笔记、怎么划重点"的技巧合集，而是用同一个学习力思想体系回答"学习如何发生"：
- **《学会如何学习》**用 120 个真实学习案例的点评，反复示范同一件事：学习发生的标志不是"听懂了"，而是"用起来了、自己被改变了"。
- **《反向学习》**给出了更系统的理论框架——学习力的**五项修炼**（反向学习→参考答案思维方式→聚焦→模式化学习→深层迁移），每项修炼替代一个妨碍学习的坏习惯。

## 它们从哪来

作者刘澜是学习力研究者和顾问。

- 《学会如何学习》的 120 堂"轻课程"源于一个学习社群（《学习之美》学习圈）：每天一位成员分享自己的学习收获或困惑，刘澜做语音点评，再把其中有代表性的案例改写成课。所以这本书的"课文" = 真实人的真实卡点 + 作者的参考答案。
- 《反向学习》是同一套思想的系统化表述，提出学习力的五项修炼框架。作者自述：如果只选最重要的一项修炼，他会选第五项"深层迁移"。

读者定位：企业管理者、创业者、职场新人，以及任何"学了很多却用不上"的成年人。

## 四问学习法：全书的脊柱

八个 skill 里七个都挂在它身上，因为它是书里把"用起来"讲得最透的一条线：

1. **听** —— 如实接收，先说清"我听到了什么"（不是急着评判）。
2. **找** —— 先发现模式，再在模式指导下找原因 / 答案（对应 `faxian-moshi`）。
3. **变** —— 用参考答案思维松动一个旧想法（对应 `cankao-daan-siwei`）。
4. **用** —— 落到本周一个具体的小改变（对应 `xuexi-tiwen` 的"体温"检验）。
