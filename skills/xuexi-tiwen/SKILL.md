# 学习要有体温（认知/情感/行为三维度动起来）

## R — 原文 (Reading)

> "学习有三种体温，分别来自认知、情感和行为。学习就是新旧认知、新旧情感、新旧行为的纠缠互动，新知识由此而产生。"
> "体会就是用身体去互动……切己，就是用别人的知识（切），联系自己的实际（己）。"
>
> —— 刘澜，第73课、第89课

---

## I — 方法论骨架 (Interpretation)

学习要有体温 = 认知 / 情感 / 行为（3H：Head / Heart / Hands）三维度都要动起来，新旧之间纠缠互动才产生新知识。没有体温 = 没有真正学习发生。

**体温概念的演化**（作者明确呈现思考过程，是"注重过程而非结果"的示范）：
- v1（第24课）：两种带着体温学习的方式 —— 用身体辅助学习（听想变阶段调动身体，如站到桌子上换视角）+ 用身体落实学习（"用"阶段真正行动起来）；
- v2（第72课）：两种体温 —— 思考的体温（学到的知识跟已有知识怎么联系）+ 行动的体温（学到的知识跟行动怎么联系）；
- v3（第73-75课）：三种体温 —— 认知 / 情感 / 行为，对应 3H 模型。

**两个操作工具**（拆解汉字得来，第89-90课）：
- **切己 = 己 + 切**。"己"是举自己的例子、谈自己的实际问题；"切"是在别人的知识与自己的问题之间建立认知/情感/行为三个维度的联系。
- **体会 = 体 + 会**。"体"是身体/自己，"会"是会合、互动；体会就是用身体去互动，从认知/情感/行为三维度互动。
- 两个词"其实很像，意思差不多"——体对应己，会对应切。作者翻来覆去变着花样讲，是因为重要的事情要说三遍、说十遍。

**知识的三层境界**（第108课，借禅宗三境界）：
- 第一层：看山是山 —— 具体情境中的经验知识（碎片，看黄山是黄山）；
- 第二层：看山不是山 —— 脱离情境的抽象模式（看黄山不是黄山，是"山"）；
- 第三层：看山仍是山 —— 抽象模式与具体情境的结合（同时看到"山"和黄山），是最高级的知识，对应四问学习法中的"用"。

**困惑要有体温**（第108课）：困惑必须是自己的、来自实际问题，答案才准确；替黄山困惑得不到真成长。

## A1 — 书中的应用 (Past Application)

### 案例 1: F 同学分享要事管理"没体温"（第89课）
- **问题**: F 同学分享作者关于要事管理的主要观点，以及他在工作中的应用。作者点评：仍然存在"体温"问题——"我看到的只有对我的知识以及他的做法的介绍"。
- **方法论的使用**: 作者用"体会=体+会"框架从三维度诊断——认知上（新旧认知如何融合？）、情感上（认知融合带来的情感变化？比如对"正确的人"的认知是否引起对某个员工的情感变化？）、行为上（应有的行为与现有行为有何不同？情感在多大程度上阻碍了行为转变？）。
- **结论**: F 同学只介绍了知识和做法，三个维度的互动都没讲。体温不足 = 真正的学习没发生。
- **结果**: 作者借此提出"体会就是用身体去互动"，从认知/情感/行为三维度互动——"真正的学习，的确很难。你需要在认知、情感和行为三个维度上动起来。"

### 案例 2: X 同学切己在"己"上好，在"切"上不够（第90课）
- **问题**: X 同学分享读到的一段话如何加深了他对"制定人生愿景的三环模型"的理解。作者之前肯定过他经常举自己的例子。
- **方法论的使用**: 作者用"切己=己+切"框架诊断——X 同学在"己"（举自己的例子）上做得好，但在"切"（在别人知识与自己问题间建立认知/情感/行为联系）上不够。
- **结论**: 若要求再高一些，他应该讲：自己以前对三环模型的认知是什么、存在的困惑是什么、读到的这段话是怎么解开这个困惑的。只举例子不等于切己，还要建立"新旧联系"。
- **结果**: 作者借此拆解"切己"二字，并指出"体会"和"切己"其实很像——体对应己，会对应切，是同一件事的两个说法。

### 案例 3: 两位同学困惑没体温，"替黄山困惑"（第108课）
- **问题**: 两位同学分享"我的困惑"。一位困惑"如何学习才能做到形神兼备"（举日本对西方和中国的学习为例）；另一位困惑"如何建立模式树和模式的森林"（没举任何例子）。
- **方法论的使用**: 作者用"困惑要有体温"诊断——他们"表达"出来的困惑很不相同，但"表现"出来的困惑相似：困惑没有体温。他们分享的困惑是张三的、李四的、约翰的、玛丽的，不是"我的"。
- **结论**: 学习始于困惑，但困惑要有体温，需要是你自己的、来自实际问题。三个原因：①自己的困惑更有动力学习；②困惑是变革和成长的机会，与己无关就白白困惑；③有体温的困惑才推动更高级知识产生。
- **结果**: 作者借此提出知识三层境界——看山是山→看山不是山→看山仍是山，第三层是抽象模式与具体情境的结合，是最高级知识。"如果你是一个小土山，就别去替黄山困惑。"

### 案例 4: 作者自己体温概念的演化过程（第24→72→73-75课）
- **问题**: 作者在学习群点评中逐步发展"体温"概念，从 v1 到 v2 到 v3。
- **方法论的使用**: 作者不掩饰概念的不完备，明确呈现演化过程：v1 两种带着体温学习的方式→v2 两种体温（思考+行动）→v3 三种体温（认知/情感/行为，对应 3H）。每一版都指出前一版的不足（v1 没讲清"联系已有知识"；v2 漏了情感维度）。
- **结论**: 这是"注重过程而非结果"策略的自我示范——读者看到"我如何推导"，而不只是结论。
- **结果**: 前言明示"我想要给你呈现一个有体温的思考过程"，本书"有体温"不仅指同学们的体温，更指作者的体温。这个案例本身就是"体温"的最好示范。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 读完一本书/一节课后，想检验自己是否真学到了（而非只是"听懂了"）。
2. 分享学习收获时，怀疑自己讲的是"知识博物馆"而非真学习。
3. 觉得自己"懂了"但用不起来，想诊断卡在哪一维度。
4. 困惑很多、求知欲很强，却看不到成长。
5. 作为老师/管理者，想判断学生/下属的学习是否真发生。

### 语言信号 (用户的话里出现这些就应激活)

- "学了很多像没学到" / "不知道算不算真学到" / "怎么知道学习真发生了"
- "我懂了但用不起来" / "听懂了讲不出来" / "会背但不会用"
- "困惑很多但没成长" / "替别人困惑" / "替黄山困惑"
- "这本书我读完了，但讲不出来什么"
- "how do I know learning is real" / "am I really learning" / "learning with temperature"

### 与相邻 skill 的区分

- 与 `si-wen-xuexi-fa`（四问学习法）的区别：那个是流程工具（听→找→变→用），本 skill 是状态检验（认知/情感/行为三维度是否都动起来）。二者配合——四问学习法是骨架，体温是血肉。
- 与 `faxian-moshi`（发现模式）的区别：那个找背后的模式（第二步"找"的深化），本 skill 检验模式是否真内化（是否结合自己的情境，达到第三层境界"看山仍是山"）。
- 与 `cankao-daan-siwei`（参考答案思维方式）的区别：那个是思维底座（不把答案当标准答案），本 skill 是学习状态的检验（学习是否真发生）。前者管"怎么想"，后者管"是否真学到"。

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应引导用户按以下步骤走：

1. **三维度体检**
   - 认知：新知识与我已有的"我知道"如何联系？是颠覆、补充还是修正？
   - 情感：这种认知融合带来的情感变化是什么？（困惑、好奇、喜悦、抗拒？）
   - 行为：应有的行为与我以前/现在的行为有何不同？情感在多大程度上阻碍了行为转变？
   - 完成标准：三个维度都能给出具体回答，而不是只介绍知识和做法。若某一维度答不出，说明该维度体温不足。

2. **切己检查**
   - 己：我举了自己的例子吗？谈的是自己的实际问题吗？
   - 切：我在别人知识与自己问题间建立了认知/情感/行为联系了吗？讲了以前自己的认知和困惑吗？别人的知识如何解开我的困惑？
   - 完成标准：能说出"我以前以为 X，听了 Y 之后发现 Z，所以现在打算 W"这样的完整链条，而非只复述别人的话。
   - 判停条件：若只能讲遥远例子（谷歌、刘备）或只罗列书本内容，提示这是"天文学家式学习"，回到第一步先找自己的实际问题。

3. **三层境界定位**
   - 第一层（看山是山）：我学到的还只是具体情境中的经验知识（碎片）吗？
   - 第二层（看山不是山）：我能抽象出脱离情境的模式吗？
   - 第三层（看山仍是山）：我能把抽象模式与自己的具体情境结合起来吗？
   - 完成标准：能定位自己处在哪一层，并能说出迈向下一层需要做什么。最高目标是第三层。

4. **体温不足时的补救**
   - 认知维度不足：问"这和我已有的什么知识有联系？"
   - 情感维度不足：问"这个知识让我对什么人/事产生了什么新的情感？"
   - 行为维度不足：问"我以前怎么做，现在打算怎么做？"
   - 通用补救：举自己的例子 + 联系实际问题。
   - 完成标准：能用第109课的两个提升体温的问题回答——"这对我有什么用？""我能举个自己的例子吗？"

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯信息查询**：用户只是想知道某个事实（"这本书作者是谁"），不需要体温检验。
- **机械记忆**：背单词、记公式等机械记忆任务，体温是负担。
- **初次接触新领域**：还没有"旧"可对照，三维度互动无从谈起——先做垒基学习（见 `moshi-jujiao-xuexi`）。
- **情绪倾诉**：用户在情绪中，需要共情而非流程化诊断。

### 作者在书中警告的失败模式

- **学习没有体温**（ce16，第72/75/89/108课）：只介绍知识和做法，没有讲新旧认知如何联系、情感如何变化、行为如何转变。"像看体温计度数而非感受体温"——看到知识和做法的罗列，却看不到一个活生生的学习过程。作者明确说："我怀疑是否真的有学习发生，至少，我怀疑是否有深刻的学习发生。"
- **不切己**（ce17，第90/113课）：要么只谈抽象问题不联系自己（"己"不够，如困惑"如何提出好问题"这种宏大问题），要么只讲自己的情况不联系抽象知识（"切"不够，变成"空谈实际"）。困惑过于宏大不切己，或过于具体缺少"切"的动作。作者引述："永生永世不谈不切己的问题。永生永世不以不切己的方式谈论问题。"
- **假顿悟**（ce29，第26/91课）：把听到一个"知识金句"的兴奋当成顿悟。经常"顿悟"但其实都是假的——"你不是醍醐灌顶，你是糨糊灌顶。"学到的是谈资而非改变。真正的顿悟必须能"用"（改变认知或行动），不能"用"的顿悟不是真顿悟。
- **天文学家式学习 / 空谈理论**（ce03，第18/25/26课）：只谈离自己很遥远的例子（谷歌、刘备）或只谈书本上说了什么，不联系自己的实际。"讨论半天别人不知道他是做什么的"——这就是天文学家的证据。作者判断标准：如果你跟别人讨论学习，讨论半天后别人不知道你的职业，那么你还不太会学习。

### 作者的盲点 / 时代局限

- **"体温"是作者独创的隐喻，非学术术语**——借身体温度隐喻学习中"新旧互动"的状态。在学术语境中需用日常语言解释，不能当作严谨概念套用。作者自己承认这个概念在书中经历了三次演化，并非定型结论。
- **3H 模型借用自英文学习理论界**，作者区分三个不同版本（分别发展 / 同时调用 / 共同转变），但未充分引用原始文献，学术严谨性不足。作者自己也说"我读了几篇关于学习的 3H 模型的论文，发现它们说的不完全是一回事"。
- **案例多为企业管理者/创业者/职场人**，对学生、蓝领、自由职业者、退休人群的适用性需用户自行迁移。
- **对学校教育的简化批评**：作者多处将学校教育简化为"为考试"，忽视学校在系统知识结构、社会化、批判思维训练上的价值。

### 容易混淆的邻近方法论

- **心流（flow，契克森米哈伊）**：描述专注投入的状态，强调技能与挑战匹配；体温不强调专注，而强调新旧认知/情感/行为的"纠缠互动"。心流是状态描述，体温是学习发生机制。一个人可以心流地背完一本书，但若没有新旧互动，仍是没体温的学习。
- **具身认知（embodied cognition）**：学术理论，强调身体在认知中的核心作用；体温是学习论概念，包含身体但不限于身体（情感也是体温的一部分）。作者 v1 版本接近具身认知，v3 版本已扩展到认知/情感/行为三维度。
- **库伯学习圈（Kolb's cycle）**：具体经验→反思观察→抽象概念化→主动实践；与本 skill 的三维度有交集，但库伯强调循环（四阶段），体温强调"新旧纠缠互动"（三维度同时发生），不是阶段而是状态。
