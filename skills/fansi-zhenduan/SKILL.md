# 反思与诊断（三环反思 + 负面情绪信号 + 学习导向诊断）

## R — 原文 (Reading)

> "各种负面情绪告诉我们：事情不太妙，需要改变现状。利用负面情绪来学习：第一，捕捉具体负面情绪，明确学习方向；第二，把负面情绪'正面化'；第三，应用三环学习模型反思。"
>
> —— 刘澜，《学会如何学习》第116课

> "反向学习指有意遗忘头脑中已有的知识。所谓反向学习，就是减法式学习，是去除头脑中错误、过时、低效的知识。反向学习的重点，不是单点的碎片知识，而是成套的心智模式。"
> "反向学习不仅要'卸载'已有的坏习惯，而且要'安装'新习惯来替代它们。"
> **四个妨碍学习的坏习惯 → 四项修炼的替代**：标准答案思维方式→参考答案思维方式；发散式学习→聚焦；碎片化学习→模式化学习；学用脱节→深层迁移。
> **反向学习的口诀**："我要改变什么？"
>
> — 刘澜，《反向学习：学习力的五项修炼》第一章"反向学习"（机械工业出版社，2025）

## I — 方法论骨架 (Interpretation)

本 skill 是"学习卡住后"的补救工具，含三个子工具，可单独用也可组合用。

在《反向学习》的五项修炼体系中，它对应**第一项修炼"反向学习"**——减法式学习，卸载坏习惯再安装新习惯。口诀是"**我要改变什么？**"《反向学习》系统列出了四个妨碍学习的坏习惯及其修炼替代：标准答案思维方式→参考答案思维方式；发散式学习（弥补短板、全面发展）→聚焦；碎片化学习→模式化学习；学用脱节（学而不用/学而乱用）→深层迁移。本 skill 的三个子工具就是帮你执行反向学习的诊断流程。

1. **三环学习模型**（借阿吉里斯双环学习扩展）——行动时带着三层假设：这个行动能实现目标（单环）/这个目标值得追求（双环）/我追求目标的底层心智模式合理（三环）。反思时分别检视三层，由外向内越来越难，**心智模式的改变最难也最重要**。多数人卡在单环（换行动），不愿改目标，更不愿动心智模式。
2. **负面情绪是学习信号**（处方性主张，非描述性研究）——进化论依据：肉体痛苦感是生存必需（无疼痛感的人会把自己打坏），情绪痛苦同理=需要改变的信号。三步：①捕捉具体情绪明确方向（焦虑/无聊/沮丧指向不同层面）②正面化（"我不是焦虑，我是在用心思考如何避免出错"）③三环反思。注意：这是"应然"处方——不管大多数人被负面情绪击倒，我们应这样给自己开方。
3. **学习导向诊断**（七种/八种）——原模型（英国大学生）：学术(内/外)+职业(内/外)+个人(内/外)+社交共七种；作者改造为职场八种：求知(内/外)+工作(内/外)+自我(内/外)+关系(内/外)。**是分析模式非描述模式**——一人往往混合多种导向。不同导向指向不同学习策略，可诊断反面镜：博物馆式学习（多为工作外导向，学得广不深）、贪官式学习（学了为教不为自己用）、天文学家式学习（学术外+自我外导向，谈资而非工具）。

配套：**老师解决两种困惑**（第104课）——表达出来的困惑（学生直接说出口）vs表现出来的困惑（学生不自知但行为暴露），可迁移为自我诊断："我表现出来的、自己却没意识到的困惑是什么？"

## A1 — 书中的应用 (Past Application)

### 案例 1: X 同学"空谈实际"——三环反思的缺席（第115、116课）

- **问题**: X 同学详细分享了一次失败的销售案例，希望大家探讨为什么失败。他一直在讲故事、讲琐碎行动，但讨论半天没有成果。
- **方法论的使用**: 作者用"三环学习模型"诊断——X 同学针对案例讲了很多琐碎的**行动**，但缺乏对**目标**和**心智模式**的提炼。即卡在单环（复盘行动），没走到双环（这个销售目标本身值不值得追求）和三环（我对销售/客户的底层心智模式是不是出了问题）。同时指出他缺元认知："我当时是怎么想的？"——没有把背后朴素的理论提炼出来。
- **结论**: 失败案例若只讲行动不讲目标和心智模式，就不是"从失败中学习"，只是"关于失败的困惑"——正如"干"不等于"干中学"。
- **结果**: 作者借此区分"空谈理论"（E 同学）和"空谈实际"（X 同学）两种反面，并提出"联系实际"必须有双方——实际和理论，缺一不可。

### 案例 2: C 同学"提不出好问题"——学习导向诊断（第111-112课）

- **问题**: C 同学的困惑是"自己为什么提不出好问题"。他自己总结了三点原因：懒得思考、没有复盘习惯、没有提问框架。归因于"懒"。
- **方法论的使用**: 作者用"学习导向"诊断——C 同学是培训师，他学绝大多数内容想的是"怎么讲给别人听"而非"我怎么用"，属于**工作导向（外在动机型）**，本质是博物馆式学习+贪官式学习。这解释了三点表面原因：①之所以直接给答案而非提问，是因为他从"博物馆"搬出藏品当答案，像讲解员而非提问者；②没有复盘习惯是因为博物馆里只有展品没有可反思的"看家模式"；③有提问框架但没真用，因为框架也是展品。
- **结论**: 归因于"懒"使方案表面化且缺乏操作性；归因于"学习导向"则指向可改变的结构——从外在动机转向内在动机，从"学了为教"转向"学了为用"。
- **结果**: 作者给出非常具体的方子（因为 C 的困惑有体温），并借此示范"困惑有体温，答案才准确"。

### 案例 3: G 同学"至暗时刻"——负面情绪作为信号+聚焦三环（第117-118课）

- **问题**: G 同学遭遇降薪或失去职位（至暗时刻），情绪不好，困惑表达不清，大概是自己以前学习取得了职业成就，现在形势复杂有深深的无力感，该怎么学习。
- **方法论的使用**: 作者先用"负面情绪是学习信号"——情绪过于强烈时难以学习，先平复；平复后把至暗时刻作为信号："事情不太妙，我该怎么改？"然后用三环反思+聚焦三环：①也许什么都不用改（逆境与他无关，是运气）；②但他对逆境的反应说明心智模式可改——把逆境作为学习机会、把成功更多归因于运气（而非能力）；③未来学习聚焦"擅长×热爱×机会"三环交叉地带，而非泛泛地学。
- **结论**: 负面情绪是入口，三环反思是路径，聚焦三环是出口。G 同学以前可能享受行业红利却归因为自己能力强，所以受打击大——这是心智模式问题。
- **结果**: 作者同时指出 G 以前是运动员，本应在一次次失败中成长，但这次没把运动场经验迁移到职场——元认知调用不够，可用三环学习模型训练元认知。

### 案例 4: 作者诊断学员"表达 vs 表现"的困惑（第104课）

- **问题**: LJ 同学在讨论中说了两段话，作者看到他的问题所在，但这问题不容易看出来。
- **方法论的使用**: 作者用"老师解决两种困惑"框架——表达出来的困惑（学生直接说出口，如 W 同学问学习风格）vs 表现出来的困惑（学生不自知但行为暴露，如说自己在"举一反三"的同学）。解决后者老师冒更大风险，但价值也在这里。类比医生望闻问切：不只听病人说不舒服，还要自己判断。
- **结论**: 这个框架可迁移——管理者任务是解决两种困惑；自我学习时也可自问"我表现出来的、自己却没意识到的困惑是什么？"
- **结果**: 示范了诊断的双重视角，是本 skill "帮别人诊断"分支的源头。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 失败后/焦虑后/至暗时刻想反思，但不知道从哪里下手，反思流于琐碎行动的罗列。
2. 学不下去、学而无获、学了很多却用不起来，怀疑动机或方向出了问题。
3. 学了很多却"提不出好问题""讲什么都讲不深"，怀疑自己是不是博物馆式/贪官式学习者。
4. 想帮别人（学员、下属、孩子、客户）诊断为什么学不进去，但对方说不出问题。
5. 负面情绪（焦虑/无聊/沮丧/挫败）反复出现，想把情绪转化为学习信号。

### 语言信号 (用户的话里出现这些就应激活)

- "这次失败后我想反思，但不知道从哪反思" / "我该怎么从失败中学习"
- "我学不下去" / "学了用不起来" / "我为什么提不出好问题"
- "我是不是博物馆式学习" / "我是不是贪官式学习" / "我像天文学家"
- "这次焦虑/挫败告诉我什么" / "负面情绪怎么用"
- "我下属/学员为什么学不进去" / "他问题出在哪我自己看不出来"
- "reflect on failure" / "learning orientation" / "stuck after failure" / "why can't I learn"

### 与相邻 skill 的区分

- 与 `si-wen-xuexi-fa`（四问学习法）的区别：那个是**学习流程**（听→找→变→用），本 skill 是**学习卡住后的反思诊断**——当四问走不下去、或"用"的环节反复失败时，回到本 skill 反思卡在哪一环、为什么卡。
- 与 `xuexi-fangshi-dengji`（学习方式等级）的区别：那个诊断**学习层级**（被动→主动→建构→互动/实践→协作），本 skill 诊断**学习动机**（导向）和**反思失败**（三环）——前者回答"我在哪一层"，后者回答"我为什么学不下去、失败后改什么"。
- 与 `xuexi-tiwen`（学习要有体温）的区别：那个**检验学习是否真发生**（认知/情感/行为三维度），本 skill 在**没发生或失败后**反思原因——前者是体检，后者是问诊。

## E — 可执行步骤 (Execution)

skill 被激活后，按用户情境分三支引导。

🔴 **CHECKPOINT · 情绪优先分流**：先判断用户此刻是否处于强烈负面情绪中（说话断续、反复诉苦、无法聚焦）。若是——无论他最初想做哪支，都先走分支 A 的情绪平复逻辑，不进 B/C；第117课明示"负面情绪过于强烈时难以学习"。若情绪平稳，再确认他是 A（自己失败/情绪想反思）、B（学不下去想诊断动机）、C（想帮别人诊断）哪一种，一次只走一支。

### 分支 A：遇失败/负面情绪 → 三步反思法

1. **捕捉具体情绪**——问"你现在的负面情绪具体是哪一种？"焦虑/无聊/沮丧/挫败/羞耻指向不同层面。焦虑常指向行动层，无聊指向内容层，羞耻指向心智模式层。
   - 完成标准：能说出一种具体情绪名词，而非"心情不好"。
2. **正面化情绪**——引导把"我不是 X，我是在 Y"造句。如"我不是焦虑，我是在用心思考如何避免出错"。避免被情绪击倒或沉溺其中。
   - 完成标准：产出一个中性/积极的重述句，且用户认可这不是自欺。
   - 判停条件：若情绪仍过于强烈（用户说话断续、反复诉苦、无法聚焦），先停止反思，建议平复后再回来——第117课明示"负面情绪过于强烈时难以学习"。
3. **三环反思**——由外向内依次检视：
   - 行动（单环）：这个行动能否实现目标？要不要换行动？
   - 目标（双环）：这个目标本身值得追求吗？要不要换目标？
   - 心智模式（三环）：我追求这个目标的底层心智模式合理吗？（如"成功主要靠能力"vs"成功主要靠运气"）★ **重点**
   - 完成标准：每一环都给出一句明确的判断，且至少在三环层产出一个"以前没意识到"的发现。若三环都"没问题"，多半是反思没到位——回到第一环再深入。

### 分支 B：学不下去/学而无获 → 学习导向诊断

1. **诊断学习导向**——让用户列出最近学的 3-5 样东西，逐个问："学这个是为了什么？"对照八种导向（求知/工作/自我/关系 × 内/外）归类。
   - 完成标准：每样东西都有一种主导导向+一种次要导向的判断。
   - 判停条件：若用户对所有东西都说"就是想学，没为什么"，提示这本身是个信号——大概率是社交/自我外导向（为了谈资而非工具），回到第 2 步用反面镜逼出证据。
2. **对照反面镜**——用三面镜子检视：
   - 博物馆式：学的东西彼此无联系、无深度，只是为了"讲给别人听"或"显得懂得多"？
   - 贪官式：学了为教别人，自己不打算践行？
   - 天文学家式：只谈遥远例子（谷歌、刘备）或只谈书本，不联系自己的实际？
   - 完成标准：明确说出自己主要落入哪面镜子（可多选），并举一个具体证据。
   - 判停条件：若用户说"我哪面都不像"，提示这是回避诊断——找最近一次"学完但没用上"的具体例子，从那一次反推落入了哪面镜子。
3. **调整导向**——从外在转内在、从社交转求知、从"为教"转"为用"。具体动作：选一样"工作外导向"的内容，转为"工作内导向"——问"这个内容怎么帮我把现在的工作做得更好？我具体用到哪里？"
   - 完成标准：产出一个导向调整的具体承诺+一个近期可执行的动作。
   - 判停条件：若用户只做口头承诺（"以后我会注意用在自己身上"）却说不出具体用到哪，提示这仍是贪官式语言，回到第 1 步重选一样内容落地。

### 分支 C：帮别人诊断 → 区分两种困惑

🔴 **CHECKPOINT · 助人意图确认**：进分支 C 前先确认用户是真想帮对方（学员/下属/孩子）诊断，还是只是来吐槽"这个人怎么学不进去"。若是吐槽——本 skill 不接，建议回到分支 B 诊断用户自己。确认是助人意图后，再走两步困惑的区分。

1. **先听表达出来的困惑**——让对方说他自己的困惑。注意：若困惑过于宏大（"怎么提出好问题"）或过于琐碎（全是故事），都不算合格的困惑。
2. **再观察表现出来的困惑**——自问"他本人不觉得是问题、但行为暴露的问题是什么？"如：说自己在"举一反三"实则没真懂、问问题是为炫耀而非求知、教别人的东西自己不用。
3. **诊断后开方**——像医生治病，针对具体情况开方（王阳明"因病开方"）。不要给一般性答案——"困惑有体温，答案才准确"。
   - 完成标准：给出一个针对对方具体表现的具体判断+一个具体建议，而非通用模型。若对方是"装睡的学习者"（ce14，假装学习只为社交/炫耀），明示本 skill 无法叫醒——只能等"社会的毒打"或放弃。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **情绪太强烈时**：第117课明示"负面情绪过于强烈时难以学习"。先平复（运动、倾诉、睡眠），再回到三步反思。本 skill 不是危机干预工具。
- **纯技术问题**：代码 bug、设备故障、纯知识盲点，不需要三环反思——直接查文档、问专家。三环反思用于"反复出现同类失败"或"行动-目标-心智模式层的问题"，不是所有问题。
- **单纯记忆背诵受挫**：背不下单词、记不住公式，是方法/练习量问题，不是动机或心智模式问题。用 `si-wen-xuexi-fa` 或专门记忆法。
- **首次尝试某事失败**：第一次失败往往是信息不足或技能生疏，反思到"行动层"即可（换方法）。只有**同类失败反复出现**才需要走到目标层和心智模式层。

### 作者在书中警告的失败模式

- **ce01 博物馆式学习**：学了很多模型但彼此无联系、无深度，只为讲给别人听或显得懂得多。面对任何问题都能从"博物馆"搬出藏品当答案，但不深究、不真用。提不出好问题（像讲解员而非提问者）。本 skill 分支 B 的核心诊断对象。
- **ce02 贪官式学习**：学了只为教别人，自己不打算践行。教营销但不真用营销，教领导力但不学领导力。商学院院长、培训机构、老师都可能陷入。本 skill 用学习导向诊断（工作外导向）识别。
- **ce03 天文学家式学习**：只谈离自己很遥远的例子（谷歌、刘备）或只谈书本上说了什么，不联系自己的实际。讨论半天别人不知道他是做什么的。学术外+自我外导向的混合。本 skill 用"联系实际的三条路"作解药。
- **ce04 空谈实际**：只讲故事、讲失败案例、讲琐碎行动，但不提炼理论、不调用模式。把"举自己的例子"变成"沉浸在自己的故事里"。"联系实际"必须有双方——实际和理论，缺一不可。X 同学的案例即为典型。本 skill 分支 A 的三环反思专门处理——把行动层的琐碎复盘上升到目标和心智模式层。
- **ce14 装睡的学习者**：假装学习——问问题不是真想知道答案，而是为了炫耀或社交。只有"我知道我知道"，没有真正进入"我知道我不知道"。无法叫醒，只能等"社会的毒打"或老师用"大棒子"打。本 skill 分支 C 遇此类应明示无能为力。

### 作者的盲点 / 时代局限

- **"负面情绪是信号"是处方性主张，非普适真理**：作者明示这是"应然"处方（不管人们怎么做，我们应这样开方），不是描述性研究。但实际使用中，严重负面情绪（抑郁、创伤后应激）需要专业心理干预，强行"正面化"可能造成二次伤害——本 skill 不替代心理咨询。
- **三环学习模型借用阿吉里斯并本土化**：阿吉里斯原模型是双环（行动/心智模式），作者加"目标"成三环。作者自承在第118课对斯坦诺维奇反思心智的解释"很可能不符合他的原意，是同化学习而非顺应学习"——借用有扭曲，严谨场合需回原文献。
- **学习导向来自英国大学的研究样本**：原模型基于一所英国大学的大学生样本，作者改造为职场八种。样本窄、文化特定，对中国职场、自由职业者、蓝领、退休人群的适用性需用户自行验证。
- **案例样本窄**：作者案例来自一个微信群30位正式生+10余旁听生，多为企业管理者/创业者/1位博士生。X/C/G 三位同学的诊断案例都偏管理培训场景。

### 容易混淆的邻近方法论

- **ABC 认知行为疗法**（Albert Ellis）：类似处——都承认"情绪-认知-行为"联动，都强调改变认知来改变情绪。区别：ABC 关注"非理性信念"的认知扭曲纠正，是心理治疗工具；本 skill 的三环反思关注"行动-目标-心智模式"三层假设的检视，是学习反思工具。ABC 的"信念"≈本 skill 的"心智模式"，但 ABC 不强调"目标层"的反思。
- **成长型思维**（Carol Dweck）：类似处——都强调心智模式/信念的改变。区别：成长型思维是单一心智模式（能力可增长 vs 固定），本 skill 的心智模式层是开放的——可能是"成功靠能力还是运气""失败是坏事还是信号"等任何底层假设。成长型思维可视为本 skill 三环反思中"心智模式层"的一个特例。
- **PDCA/PDSA**（戴明环）：与三环学习模型形似，但 PDCA 是质量管理循环（plan-do-check/study-act），重在"行动-反馈-修正"（单环）；三环学习模型多出"目标层"和"心智模式层"的反思。作者指出四问学习法是 PDSA 中 P（plan）的形成过程，本 skill 是 PDSA 中 S（study）的深化——尤其走到双环和三环时。
