# 学习方式等级模型（被动→主动→建构→{互动,实践}→协作）

## R — 原文 (Reading)

> "一个比较完整的学习方式等级模型（按照有效程度从低到高）是这样的：最底层是被动学习，往上是主动学习，再往上是建构学习，再往上则左右分叉，并列为互动学习（讨论式学习）和实践式学习，再往上则左右合流为协作式学习。"
>
> — 刘澜，第107课

---

## I — 方法论骨架 (Interpretation)

学习方式有等级，越高级越有效，也越难。从低到高五级：**被动→主动→建构→{互动, 实践}→协作**。后者包含前者，并调动更多认知手段——这正是它更有效的原因，也正是它更难的原因。

- **被动学习**：只接收，如听课、听书、看小视频。
- **主动学习**：用自己的话复述、提问、提炼要点。
- **建构学习**：用自己的语言和角度重组，把单点连成网络。
- **互动学习**：与他人讨论（TALKS）；**实践式学习**：干中学，带假设+反馈+修正。两者都比建构高级，但路径不同。
- **协作式学习**：一群人合作做一个项目，=互动+实践。最难，潜力满分 200，但很可能只得 80。

作者对季清华 ICAP（被动/主动/建构/互动）的升级是补入**实践式学习**和**协作式学习**——前者针对学校教育不研究"干中学"的盲区，后者点破"行动学习项目难做"的根源：它是最高级学习方式，参与者往往连基本层级都没掌握。

配四种"我知道"元认知诊断学习准备度：①我知道我知道；②我知道我不知道（第一种，来自对知识的思考，更高级）；③我知道我不知道（第二种，来自对实践的思考）；④我不知道我不知道（盲区）。学习路径：第二种"不知道"→"知道"→第一种"不知道"（更高级困惑）。"运用自如"往往卡在低层"我知道我知道"，未进入"我不知道"。

## A1 — 书中的应用 (Past Application)

### 案例 1: 作者用等级模型诊断"听书是低效学习"
- **问题**: W 同学把听十来分钟音频解读当作读书的替代，自以为高效。
- **方法论的使用**: 作者用季清华模型诊断——听书"主要是被动学习，想要加上一些主动学习，是可以的，但能加上的有限"。读书可停下来想、画线、写批注（建构）、甚至联系作者（互动），听书时很难做到。
- **结论**: 学习方式的有效性由它调动的认知手段多少决定。听书的天花板就是被动学习+少量主动学习，而读书可上探到建构甚至互动。
- **结果**: 处方——听书可作为补充，但不能作为主要学习手段；想真正吸收一本书，必须回到读书。

### 案例 2: JK 同学"互动积极但实践难"
- **问题**: JK 同学在群里讨论所学知识（互动学习）比较积极，但难以把所学付诸实践。
- **方法论的使用**: 作者用升级版等级模型诊断——互动学习已是较高级别，但实践式学习更难，"很可能它比互动学习更加有效一些，因此也更加难一些"。
- **结论**: 从互动到实践不是自然发生，而是再上一个台阶。互动学习积极≠实践能力强。
- **结果**: 处方是带着四问学习法明确假设去干中学（实践式学习=验证并修正行动/目标/心智模式三层假设），让"干"变成"干中学"。

### 案例 3: S 同学"运用自如"实为低层"我知道我知道"
- **问题**: S 同学说自己对数个管理模型"了然于胸、运用自如"，特别提到管理者三大能力模型。
- **方法论的使用**: 作者用四种"我知道"诊断——S 同学处在低层次的"我知道我知道"，而作者自己因掌握更深才产生她没有的"我知道我不知道"。"她对这些模型的'我知道'还在知识的圆圈之内，而我的'我知道'可能更接近圆圈的边界。"
- **结论**: "运用自如"是相对以前的自己而言，面对未来的自己应保持第四种谦虚。掌握得越浅，越容易觉得掌握；掌握得越深，越能发现自己的"不知道"。
- **结果**: 处方——保持愚蠢才能保持饥饿，从"我知道我知道"跨越到"我知道我不知道"才是真升级。

### 案例 4: 行动学习项目为何难做
- **问题**: 企业培训中的"行动学习"项目（一群人合作解决一个实际工作问题）普遍推不动、效果差。
- **方法论的使用**: 作者用等级模型诊断——行动学习就是协作式学习，是最高级学习方式，"包括了所有的学习方式。而那些参与者很可能连比较基本的学习方式都没有掌握好"。
- **结论**: 协作式学习潜力满分 200，但参与者层级不够，实际只得 80。越有效的学习方式越难，倒过来说也成立。
- **结果**: 处方——别直接上协作式学习，先评估参与者是否已掌握被动/主动/建构/互动各层级，逐级升级。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 学了很多（听课/读书/听书/看视频）却感觉没进步，想诊断自己的学习卡在哪一层。
2. 想从被动学习升级到主动/建构/互动学习，但不知道怎么升。
3. 团队推动行动学习项目/协作学习，但效果差，想知道为什么。
4. 自以为掌握某个模型/方法，被点破是"运用自如"的虚假掌握。
5. 纠结该选哪种学习方式（听书 vs 读书、看视频 vs 读原著、自学 vs 讨论 vs 干中学）。

### 与相邻 skill 的区分

- 与 `si-wen-xuexi-fa`（四问学习法）的区别：那个是**单次学习流程**（听→找→变→用），本 skill 是**学习方式的层级诊断**（被动→主动→建构→{互动,实践}→协作）。四问学习法可在任一层级使用，但不同层级效果不同——本 skill 诊断你在哪层。
- 与 `xuexi-tiwen`（学习要有体温）的区别：那个检验**学习是否真发生**（认知/情感/行为三维度），本 skill 诊断**学习在哪个层级**。体温是"有没有动起来"，层级是"动到哪一级"。
- 与 `hudong-xuexi-talks`（互动学习 TALKS）的区别：那个专门展开互动学习的操作工具，本 skill 把互动学习定位在等级模型中——当你想从建构升级到互动时，先来本 skill 定位，再去 TALKS 取工具。

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应引导用户按以下步骤走：

1. **诊断当前学习在哪个层级** ★ 最关键
   - 让用户描述一次具体的学习活动（如"我听了一节 20 分钟的课"），对照五级模型判断：被动（只听）/主动（复述提问）/建构（用自己的话重组）/互动（与人讨论）/实践（干中学带假设）/协作（一起做项目）。
   - 同时用四种"我知道"诊断准备度：是"我知道我知道"（舒适区）、第二种"我不知道"（实践困惑）、"我知道"（已掌握）、还是第一种"我不知道"（高级困惑）？
   - 完成标准：能说出"我的学习在 X 层级，我的'我知道'状态是 Y"。
   - 🔴 检查点：若用户描述的学习活动无法归入任一层级（既不像被动也不像主动），先别急着定级——回到"具体描述一次学习活动"重做，不要硬套层级。若用户把听书说成"学习"，明确点出它主要是被动学习，天花板有限，不能替代读书。

2. **识别可升级的下一级**
   - 问："下一级是什么？为什么你现在升不上去？"
   - 完成标准：找到一个具体的、相邻的升级目标，而非跨越多级（如不要从被动直接跳到协作）。
   - 判停条件：若用户想从被动直接跳到协作（跨多级），拦住——要求先定一个相邻升级目标（如被动→主动），逐级爬升。协作是最高级学习方式，没掌握低层直接上协作必失败（见案例4）。

3. **用对应手段升级**
   - 被动→主动：复述要点、提三个问题、画线写批注。
   - 主动→建构：用自己的话重组；把单点连成网络；事前建模（听讲前先建构自己的模型）。
   - 建构→互动：找人讨论，调用 `hudong-xuexi-talks` 的 TALKS 模型。
   - 建构→实践：带假设去干中学，用三环学习模型（行动/目标/心智模式）反思。
   - 互动+实践→协作：一群人合作做一个真实项目。
   - 完成标准：产出一个针对当前层级的、具体的升级动作。
   - 判停条件：若用户列出的升级动作仍停留在当前层级（如"我要多听几遍"来从被动升级——听再多也是被动），指出这是原地打转，回到对应手段重新选一个真能调动更多认知的动作。

4. **警惕虚假升级**
   - 自问："我是不是在'运用自如'的舒适区？"——掌握得越浅越容易觉得掌握。
   - 自问："我是不是只调动了一种认知手段？"——VARK 把自己"定型"就是自我设限。
   - 完成标准：能说出一个自己尚"不知道"的具体点（从"我知道我知道"进入"我知道我不知道"）。
   - 🔴 检查点：若用户说不出"不知道"的点，判定还在低层"我知道我知道"的舒适区——回到第1步重新诊断，不从"虚假升级"直接进入协作处方。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯信息查询**：用户只想知道某个事实（"ICAP 是谁提出的"），不需要层级诊断。
- **单纯记忆背诵**：背单词、记公式等机械记忆任务，等级模型是负担。
- **情绪倾诉/安慰**：用户在情绪中，需要的是共情而非层级分析。
- **单次学习流程的执行**：用户想完成一次"听→找→变→用"，用 `si-wen-xuexi-fa`，不要混入层级诊断。
- **学习是否真发生的检验**：用户想判断学习有没有真发生，用 `xuexi-tiwen` 的三维度体温模型。

### 作者在书中警告的失败模式

- **小视频学习**（ce06）：媒介即讯息——小视频的形式（几分钟、娱乐化、注意力分散）决定它只能传递数据和信息，不能传递模式化知识。是典型的被动学习且无法升级。
- **听书是低效学习**（ce07）：听书"主要是被动学习，想要加上一些主动学习，是可以的，但能加上的有限"。听书时无法停下来想、画线、写批注、联系前后内容。
- **装睡的学习者**（ce14）：只有"我知道我知道"，没有真正进入"我知道我不知道"。问问题是为了炫耀或社交，不是为了学习。无法叫醒，只能等"社会的毒打"。
- **"运用自如"的虚假掌握**（ce15）：S 同学自以为运用自如，实际是低层次"我知道我知道"。掌握得越浅越觉得掌握，掌握得越深越发现"不知道"。
- **VARK 把自己"定型"**（ce22）：说自己是"听觉型"所以只听书不读书。VARK 只关注学习的"听"这一步，且把自己看作名词/形容词而非动词。重要的是"做什么"而非"是什么"。
- **不做习题**（ce28）：读书但不做习题=没有"想"和"用"的环节=假学习。做习题是主动学习、建构学习和互动学习（与出题者互动）的过程。

### 作者的盲点 / 时代局限

- **借用季清华框架的本土化整合**：作者自承在第118课对斯坦诺维奇反思心智的解释"很可能是不符合他的原意的。在这一点上，我是在同化学习，而非顺应学习"——对 ICAP 的升级也带有同化色彩，用户使用时若需严格学术语境应回到原始文献。
- **对学校教育的简化批评**：作者将 ICAP 未纳入实践式学习归因于"其提出者主要研究学校教育"——这忽视了学校在系统知识结构、社会化、批判思维训练上的价值，对学校教育的批评偏简化。
- **协作学习"有效"是潜力非保证**：作者明确"这里的有效，说的是潜力"——协作式学习满分 200 但可能只得 80。用户不要把"高级"等同于"必然有效"，低层级学好可能比高层级没学好更有效。
- **案例样本窄**：等级模型的多处案例（JK/S/行动学习）来自企业管理者微信群，对学生、蓝领、自由职业者的适用性需用户自行迁移。

### 容易混淆的邻近方法论

- **布鲁姆认知层级**（记忆/理解/应用/分析/评价/创造）：描述**认知目标**的深度层次，本 skill 描述**学习方式**的层级。前者是"学到什么程度"，后者是"用什么方式学"。
- **DIKW 模型**（数据/信息/知识/智慧）：描述**信息**的层次，本 skill 描述**学习方式**的层级。前者是"学到的内容是什么"，后者是"学习的动作是什么"。
- **布鲁姆 + 本 skill 配合**：用本 skill 诊断学习方式层级，用布鲁姆诊断学习目标层级，两者正交。
